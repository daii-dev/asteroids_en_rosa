# Asteroids: pink edition

Proyecto universitario desarrollado para la materia de **Programación Web** 
Recreación del juego clásico arcade *Asteroids* con estética rosa usando únicamente tecnologías del lado del cliente.

## 🛠️ Tecnologías utilizadas

| Tecnología | Para qué se usa |
|------------|-----------------|
| **HTML5** | Estructura de la página, canvas y pantallas del juego |
| **CSS3** | Estilos visuales, colores, animaciones de botones y efectos de brillo |
| **JavaScript** | Logica del juego: movimiento, colisiones, niveles, puntuación |
| **Canvas API** | Renderizado del juego en tiempo real (nave, asteroides, balas, explosiones) |
